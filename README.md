# HeatEquation
## Overview

HeatEquation is a simulation which implements SPH(Smoothed Particle Hydrodynamics)-based and constriant-based heat equation. This simulation is based on CPP-Fluid-Particles<sup>[1](#footnote_1)</sup>.
